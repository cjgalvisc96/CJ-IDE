# CJ-IDE tool manifest — the SINGLE SOURCE OF TRUTH for what mise installs.
#
# Both install.sh and prune.sh read this file, so there is exactly one list to
# maintain. Format:
#   * one `mise` tool spec per line  (e.g. node@lts, go:host/pkg@latest)
#   * blank lines and lines starting with `#` are ignored
#   * `# [section]` headers group tools; install.sh installs runtimes first
#     (so the go:/npm: backends below can build)
#
# Pin a version for reproducibility by replacing @latest, e.g. `ripgrep@14.1.0`.
# Neovim is pinned to a major.minor on purpose: an unpinned jump to a new
# Neovim release is what broke treesitter once already.
